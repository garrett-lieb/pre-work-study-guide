HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes are always defined by the Beginning Element, never the closing element.
  • Inside head elements, we use metadata tags.
  • Metadata is data about a webpage that the browser uses to Effectively Serve a webpage to users.
  • Elements nesteld inside of other elements are called Child Elements.
  • Child Elements nestle together under Parent Elements.
  • Semantic HTML Elements: the name of the elements describes its contents and the role it plays on a webpage.

CSS

  • CSS stands for Cascading Style Sheet, a programming language used to define how content is presented to users.
  • There are 3 ways to style a webpage in CSS:
    1. Inline CSS
    2. Internal Style Sheet
    3. External Style Sheet (this is most common)
  • External Style Sheets keep all CSS Rules in a separate file. This makes design changes easier.
  • External Style sheets use Link Elements placed in the HTML's Head.
  • Margin properties are space Outside An Element.
  • Padding properties are space around content Inside An Element.
  • The Selector defines elements or atributes to which the rules, or declarations, will be applied.
  • Separation of Concerns is a practice by which code is separated into different files based on use.

Git

  • Git is a version control system. It is free and open source.
  • Git is how GitHub talks to and operates locally on your computer.
  • git status: checks what branch you are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git init: initialize an existing directory as a Git repository.
  • git clone [url]: retrieve an entire repository from a hosted location via url.
  • git status: show modified files in working directory, staged for your next commit.
  • git branch: list your branches. a * will appear next to your current branch.
  • git merge: merge the specified branch's history into the current one.
  • git merge [alias]/[branch]: merge remote branch into your current branch to bring it up to date.
  • git push [alias][branch]: transmit local branch commits to local repository.
  • git pull: fetch and merge any commits from the tracaking branch.

JavaScript

  • JavaScript adds INTERACTIVITY to a website. It takes a static HTML/CSS webpage and makes it active!
  • It does this, in part, through variables, strings, and organizing data types.
  • JavaScript is Case Sensetive.
  • API: aaplication programming interfaces.
  • Variables: containers that store value.
  • Comments: snippets of text added along with code.
  • Operator: a mathematical symbol that produces a value based on two values or variables.
  • Conditions: code structure used to test if an expression is true or not
  • Functions are a way of packaging a task completion that you wish to reuse. They often take arguments.
  • Arguments are bits of data a function needs to do their job (found in parentheses, separated by commas.)
  • We define a body of code as a function when it executes when you call the Function Name in your code.
  • Event/Event Handlers: Functionality requires code structures to Listen for activity and run code in response (like clicking.)
  • JavaScript is a Dynamic Language with Dynamic Types. This means variables aren't associated with a particular value type.
  • JavaScript is a Weakly Typed Language. This means it allows Implicit Type Conversions.
  • Primitive Values are important to Javascript. These are all non object type that define immutabel values.
  • Primitive Values are represented directly at the lowest level of the language.
  • Some examples of Types of Primitive Values are: Null, Undefined, Boolean (a true false binary value), Number, BigInt, and String.
  • Symbol Types: a unique and immutable primitive value that may be used as the Key of an Object Property.
  • Objects in Comp Science are a value in memory which is possibly referenced by an identifier.
  • In JavaScript, Objects are the only Mutable Values; Functions are Callable Values.
  • Control Flow: the order in which a computer executes code in a script.
  • Array: group of related data contained by a Single Variable. Arrays are fundemental functions of JavaScript.
  • For loops are ways of telling the computer to handle an array.
  • For loops contain 3 important statements
    1. Starting Point
    2. Condtion
    3. How array should Iterate OVer Each Item
  • Length is a special property of an array that indicates the number of items in the array.
